# Restaurant Order System (OOP Version) using Java

## Project Name
Restaurant Order System (OOP Version)

## Description
This project is a **Restaurant Order System** implemented using **Java Swing**. It allows users to:

- View a menu with items and prices.
- Select item quantities for an order.
- Apply discounts (percentage or fixed amount).
- Calculate taxes (SGST and CGST).
- Generate a detailed bill.
- Confirm or cancel orders.
- Maintain an order history.

The system demonstrates the use of **Object-Oriented Techniques (OOP)** in Java, including classes, objects, inheritance, interfaces, and polymorphism.

## Technologies Used
- Java SE 8+
- Swing (for GUI)
- AWT (for event handling and layout)
- Object-Oriented Programming concepts

## Features
- Menu management
- Order taking with quantity selection
- Bill generation with taxes and discounts
- Order confirmation and cancellation
- Order history tracking
- User-friendly GUI

## Installation & Running the Project
1. Make sure **Java JDK 8 or higher** is installed on your system.
2. Download the project file `RestaurantOrderSystemOOP.java`.
3. Open a terminal/command prompt and navigate to the folder containing the file.
4. Compile the code:
   ```
   javac RestaurantOrderSystemOOP.java
   ```
5. Run the program:
   ```
   java RestaurantOrderSystemOOP
   ```
6. The GUI window will appear. Use it to select menu items, apply discounts, calculate bills, confirm or cancel orders.

## Usage
- Select the desired quantity for each menu item.
- Choose a discount from the dropdown (if available).
- Click **Calculate Total Bill** to view the current bill.
- Click **Confirm Order** to finalize the order and add it to the order history.
- Click **Cancel Order** to cancel the current order.
- View all previous confirmed or canceled orders in the order history panel.
